We use Basecamp as a reference tool throughout the course. It contains the entire course curriculum as well as a calendar with all of the key dates.

We use Treehouse as a tool to learn in our own time - Treehouse has a range of courses, quizzes and code challenges we use to supplement the sessions.

We use Slack as a tool to communicate with each other. Slack is where we go to ask for help, speak to the mentors or organise meet-ups with other students.
